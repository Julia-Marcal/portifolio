Projetos

Fake-Fintech

O projeto Fake Fintech é uma iniciativa open source criada para estudar de forma prática o funcionamento de aplicações financeiras modernas. A proposta é simular um ambiente realista de fintech, integrando APIs do mercado, modelando dados econômicos e permitindo que usuários acompanhem suas finanças por meio de um sistema de rastreamento inteligente.

Além de explorar regras e conceitos fundamentais do setor, o projeto foca em boas práticas de arquitetura, performance, segurança e escalabilidade, tudo isso em um ambiente real de desenvolvimento com backend, frontend e serviços integrados.

BoilerplateX (Reusable-API)

Uma base segura, performática e reutilizável para construção de APIs robustas. Pensada para acelerar o desenvolvimento com boas práticas desde o início, está disponível em duas versões: Golang (Gin + Gorm) e PHP (Laravel), ambas com foco em performance, segurança e escalabilidade.

Ambas as versões contam com cache otimizado utilizando Redis, banco de dados relacional (PostgreSQL ou MySQL), autenticação segura via JWT/Bearer com senhas criptografadas, limitação de requisições para controle de uso da API e suporte completo a Docker e Docker Compose para facilitar o deploy e a configuração. A versão em Go também inclui monitoramento com Prometheus, documentação automática com Swagger, validação de dados com govalidator, oferecendo um ecossistema completo e pronto para produção.
